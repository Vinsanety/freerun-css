Freerun

A lightweight CSS Library

The building blocks for a design system or UI Framework.

Built with Responsiveness & Accessibility first -- all units are in rem.

Freerun includes base.css and normalize.css, customizable with only 19 CSS variables.


Typography

The font-family is set to Helvetica, Arial, sans-serif and the font-size is set to 1.125rem (18px). They are both overidable css variables.

The heading sizes are responsive, scaling on smaller viewports.

Responsive Grid

Grid system is 1-6 columns, providing 3 built-in media query classes: column-*, column-md-*, and column-lg-*. All grids must have a .container and .row that wraps the .column-*.

Freerun has three media query sizes: mobile, tablet, and desktop.

  1. Mobile column-*: 0-599
  2. Tablet column-md-*: 600-1023
  3. Desktop column-md-*: 1024+

Adjust your browser size to see it in action.

Button & Link

The button has three variations and one link. The standard button element serves as the primary, the .secondary class and .outlined class are also available.

Using an input element with type: "submit" or "reset" will also be styled as a button.

Text Link 

  <button type="button" name="button">Primary</button>
  <button type="button" name="button" class="secondary">
    Secondary
  </button>
  <button type="button" class="outlined">Outlined</button>
  <a href="">Text Link</a>

Form Elements

Built as accessible forms that are cross-browser functional and responsive. They have these customizable attributes in the base.css variable declarations: --control-border-radius, --input-border-radius, and --box-shadow.

The checkbox and radio need to be inputs nested inside of a label containing the appropariate class of .checkbox-label or .radio-label.

Best time to contact:


  <form>
    <div class="container">
      <div class="row">
        <div class="column-6 column-md-3">
          <label for="input-email-address">Email</label>

            id="input-email-address"
            type="email"
            name="input"
            placeholder="name@mail.com"
          />
        </div>
        <div class="column-6 column-md-3">

          <select id="select-contact-reason" name="select">
            <option>Option 1</option>
            <option>Option 2</option>
            <option>Option 3</option>
          </select>
        </div>
