# FreeRun CSS

FreeRun CSS is a lightweight CSS library (less than 9kb) that is fully customizable with only 19 CSS variables.

## Live Demo

Visit [FreeRun](https://freerun-css.web.app/)

## Features

- 19 CSS customizable variables.
- Built with responsiveness and accessibility first.
- Shipped with `normalize.css`.
- Consists of Typography, Grid, Buttons, Form Elements, and Lists.

## Tech Stack

- CSS
- [Normalize.css](https://necolas.github.io/normalize.css/)
